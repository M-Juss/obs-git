### Relationship
- Describe how entities interact with each other using ==Cardinalities==

**Cardinality** - help define the relationship in a numerical context particularly in a minimums and maximums

![[Pasted image 20260115092545.png]]

The 2 symbol at the end of the line can be answered as *Minimum and Maximum.*
For example we Customer and Order Entities.

A Customer can have Zero minimum orders and Many maximum orders. So We will use The Zero or many symbol.

On the other hand a specific order can have minimum of one One and  maximum of Only One Customer addressed to it. So we will use the One and Only one symbol
